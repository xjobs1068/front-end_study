과제안내

  1. 디자인에 제시된 모든 부분을 구현한다(‘과제 안내’ 부분 포함)
  2. 모든 디자인은 figma를 통해 제공된다
  3. 3/24(수)까지 완성해야하며, 과제 구현에 관한 질문은 일체 받지 않는다.
  4. 수업에서 언급되지 않은 부분이 과제에 포함될 수 있으며,
    구글링을 통해 직접 알아보는 것 또한 과제의 일부이다.
  5. 이번 과제는 반응형 웹 형태를 요구하지 않으며,
    현재 프레임의 크기인 1440x900 기준으로 평가한다.
  6. 폰트, 색상 등의 디자인적 요소는 자유롭게 변경 가능하다.
    단, 제시된 도형 간의 비율은 유지해야한다.
